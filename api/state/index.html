    battle_start_time: float | None

    Return the timestamp when battleStage transitioned to 0 (battle active), or None.

    Cached after first computation.

    state_at ¶

    state_at(t: float) -> GameState

    Reconstruct full game state at timestamp t.

    iter_states ¶

    iter_states(timestamps: list[float]) -> Iterator[GameState]

    Yield GameState for each timestamp, advancing forward.

    Optimized for sequential access (e.g., frame-by-frame rendering). Maintains running property state AND position pointers — no bisect, no copying, O(delta) per frame.

    Parameters:

    Name Type Description Default
    timestamps list[float]

    Monotonically increasing list of query times.

    		 required

    Yields:

    Type Description
    GameState

    GameState snapshot at each requested timestamp.
